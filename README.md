# GitHub Profile Finder

Aplicação web desenvolvida com HTML, CSS e JavaScript para buscar e visualizar perfis do GitHub por nome de usuário, apresentando repositórios e informações detalhadas do desenvolvedor.

O projeto foi criado com o objetivo de praticar integração com a API do GitHub, requisições assíncronas e manipulação dinâmica do DOM. Através de uma busca simples, é possível visualizar dados completos do perfile seus repositórios públicos.

Durante o desenvolvimento, foram aplicados conceitos de consumo de APIs REST, tratamento de erros e requisições HTTP assíncronas.

## 📸 Demonstração

Acesse o projeto:
🔗 https://github-profile-finder-seu-deploy.vercel.app/

## 🚀 Funcionalidades

- Busca de perfis GitHub por username.
- Exibição completa de informações do perfil (avatar, bio, followers, following).
- Listagem de repositórios públicos do desenvolvedor.
- Tratamento de erros (usuário não encontrado, limite de requisições).
